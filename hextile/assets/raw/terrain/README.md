
### layout_pure.ods

Orientations:

    0: up, 1: left&up, 2: left&down, 3: down, 4: right&down, 5: right&up

### tiles_pure.ods

Gives names to locations in tiles_pure.png.
Uses cubic coordinates; X: right, Y: up left, Z: down left.

    (0, 0, 0) is pixel (210, 160).
    x = 210 + 90 * X
    y = 160 - 40 * X - 80 * Y
